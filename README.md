# Codveda Data Analytics Internship

This repository contains all tasks completed as part of my Data Analytics internship at Codveda Technologies.  
The work is organized across three levels, covering data preprocessing, analysis, modeling, and basic NLP techniques.

---

## Levels Completed

### Level 1 – Data Cleaning & Exploratory Data Analysis
- Data cleaning and preprocessing of a housing dataset
- Exploratory data analysis (EDA) using statistical summaries and visualizations

### Level 2 – Intermediate Data Analysis
- Regression analysis for house price prediction
- Time series analysis on stock price data to identify trends and seasonality

### Level 3 – Advanced Data Analytics
- Predictive modeling using classification techniques on the Iris dataset
- Sentiment analysis on textual data using basic NLP methods
